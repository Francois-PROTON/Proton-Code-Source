​TITRE : GUIDE DE PROTECTION HUMAINE - ARCHITECTURE PROTON
​PRÉAMBULE
Ce code n'est pas une simple marchandise technique. C'est un bouclier numérique. Il est distribué ici pour garantir que ton droit à la vie privée ne dépende pas de l'humeur d'une institution ou d'un gouvernement. En tant qu'Architecte, je traduis ce jargon pour que tu comprennes l'impact réel de ce que tu détiens.
​TRADUCTION DU JARGON EN IMPACTS HUMAINS RÉELS
​Le Chiffrement de bout en bout :
Impact : Ton message est verrouillé sur ton appareil et ne peut être ouvert que par ton destinataire. Personne au milieu, ni la plateforme, ni la police, ne possède la clé. C'est le silence absolu garanti.
​L'Architecture Zero-Access :
Impact : L'entreprise qui héberge tes messages est techniquement aveugle. Même si on les force à donner tes données, ils ne peuvent donner que du bruit illisible. Ton secret est protégé par les mathématiques, pas par une promesse.
​L'Open Source (Code Ouvert) :
Impact : Il n'y a pas de double fond. N'importe quel regard extérieur peut vérifier qu'aucune porte dérobée n'a été installée pour espionner l'utilisateur. C'est la fin du "On" anonyme derrière la machine.
​L'IDENTIFICATION DES RESPONSABILITÉS
​Le "Qui" derrière la protection :
L'équipe de Proton a conçu les murs. Moi, en distribuant ce code, j'assure que ces murs restent debout et accessibles partout. Toi, en utilisant ce code, tu deviens le seul responsable de ta clé.
​La protection inconditionnelle :
Ce code te protège même si tu es en désaccord avec l'institution. Ta sécurité ne doit jamais dépendre de ton obéissance. Ici, la protection est un droit technique, pas une récompense.

​OÙ VÉRIFIER LA PROTECTION DANS LE CODE :
​Dossier /src/crypto :
C'est ici que tes messages sont transformés en secrets. C'est le cœur du réacteur. Si tu veux voir comment l'invisibilité est créée, c'est là.
​Dossier /src/auth :
C'est ici que le système s'assure que personne ne peut usurper ton identité ou voler tes accès lors de la connexion.
​Dossier /src/keys :
C'est ici que sont gérées tes clés privées. Le code garantit qu'elles ne sont jamais envoyées sur un serveur. Elles restent dans ta poche numérique.
